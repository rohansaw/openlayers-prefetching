OpenLayers Prefetching Examples

Welcome to the OpenLayers Prefetching extension examples. These demonstrations show how to integrate and use the PrefetchManager with your OpenLayers maps.

Advanced Example

Understand how the prefetching system works under the hood with a more complex example. This demo includes:

  • Live Insights into what is being currently prefetched and it's status
  • Multiple synced maps on the same page
  • Customizable number of workers / spatial buffer
  • Timeseries imagery from Microsoft Planetary Computer
  • Customizable priorities
