OpenLayers Prefetching Examples

Welcome to the OpenLayers Prefetching extension examples. These demonstrations show how to integrate and use the PrefetchManager with your OpenLayers maps.

Note: The examples currently use placeholder implementations. After implementing the actual logic in src/PrefetchManager.js, run npm run build and update the examples to import from the built module.

Basic Example

A simple demonstration showing the basic setup and controls for the PrefetchManager. Features include:

  • Enable/disable prefetching
  • Adjust prefetch distance with a slider
  • Control maximum concurrent requests
  • Manual prefetch triggering
  • Queue management

Quick Start

To get started with your own implementation:

  1. Implement the core logic in src/PrefetchManager.js
  2. Build the library with npm run build
  3. Import and use in your project: 
    import PrefetchManager from 'openlayers-prefetching';

const prefetchManager = new PrefetchManager({
  map: yourMap,
  prefetchDistance: 1,
  maxConcurrent: 4
});
